Install xampp.

start apache2 and mysql server in xampp.

place the RestApiUserAuth folder in htdocs.

import the RestApiDB database from sql database schema folder to phpmyadmin.

make sure the database name is same as .env file.

run "php artisan serve" on terminal inside RestApiUserAuth folder.

this will be base url http://127.0.0.1:8000

please refer the request reponse documentation for api endpoints.
